# rzfx

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
#文件夹说明
## common:    js,css,img等文件
## http:      接口请求和响应的js
## store:     项目需要存的状态
## router:    项目路由的设置
